Conectado a la nube ✓ Todo lo que editas se guarda al instante en Firebase y tus clientes lo ven en vivo. No necesitas descargar ni subir nada.

Activar la nube (una sola vez en Firebase)
1
En tu proyecto adminpagweb: abre Firestore Database → Crear base de datos → modo producción.
2
Abre Authentication → Sign-in method y activa Correo/Contraseña. Luego en Users → Add user crea tu correo y contraseña de administrador (con esos entras aquí).
3
En Firestore → Reglas pega esto y pulsa Publicar:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /pedidos/{id} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }
    match /{doc=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

Importante: la regla de pedidos permite que tus clientes (que no tienen usuario) registren su pedido en tu panel. Sin ella, los pedidos solo quedan en el navegador del cliente y nunca los verías aquí. Si ya habías pegado las reglas antiguas, vuelve a pegar estas y pulsa Publicar.
